Mover Ativos Financeiros entre Carteiras

Gerencie seus investimentos entre diferentes carteiras

Instruções:

  • Selecione um ou mais ativos e clique nas setas para mover
  • Move os ativos selecionados para a Carteira de Investimentos
  • Move os ativos selecionados de volta para Ativos Disponíveis
  • Use Ctrl+clique para selecionar múltiplos ativos
